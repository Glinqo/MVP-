# Mechatronics MVP Skill

Use this skill when working inside this repository on product scope, prompts, knowledge points, diagnosis rules, or demo material.

## Rules

- Keep the project lightweight.
- Prefer Markdown and JSON edits.
- Do not add services, databases, or frontend frameworks.
- Keep all examples tied to vocational mechatronics training.
- After changing scoring rules or scoring code, run `node tests/scoring.test.js`.
