# Demo Script

## 0:00 到 0:30 开场

本项目是一个面向职业教育机电一体化课程的教学实训智能体 MVP。它不做大平台，重点展示诊断、反馈、学习路径和教师视图的闭环。

## 0:30 到 1:30 学生诊断

演示输入：

```text
我做自动分拣实训时，传感器能亮，但 PLC 程序里计数不稳定，气缸动作也偶尔慢半拍。
```

说明系统识别出涉及传感器信号、PLC 计数逻辑、气动执行和故障诊断。

## 1:30 到 2:40 规则评分

展示诊断题答案进入代码节点，输出：

- 总分
- 能力节点分
- 常见错误标签
- 推荐补救节点

强调评分规则可解释，适合教师调整。

## 2:40 到 3:50 学习路径

展示系统按薄弱节点推荐：

- 复习知识点
- 一个实训任务
- 一个资源
- 一个自测题

## 3:50 到 4:40 教师视图

展示班级共性问题：

- 传感器 NPN/PNP 接线混淆
- PLC 上升沿计数理解不足
- 气缸速度调节只调程序不看节流阀

## 4:40 到 5:00 收尾

说明项目可以继续扩展到真实 LMS、设备数据采集和更细粒度的专业群能力图谱，但当前版本保持轻量、可复制、可评审。
