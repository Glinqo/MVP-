# 岗位情报演示材料

source: project_curated_demo_material

自动化生产线装调与运维技术员在新人阶段常见工作任务包括：在断电和设备安全确认后，核对 DC24V 供电，识别三线制传感器的棕、蓝、黑线，判断 NPN/PNP 输出类型，并确认 PLC 输入公共端 COM 或 S/S 的接法是否匹配。

岗位现场排查通常要求新人能够观察传感器动作灯、PLC 输入灯和在线监控状态，把现场元件、端子号、PLC I/O 地址和程序变量建立映射。遇到“传感器动作灯亮但 PLC 输入无响应”时，需要优先按电源、传感器侧、接线、公共端、输入模块、I/O 地址和程序变量的顺序定位。

在气缸不动作场景中，岗位要求新人能分层判断：先确认急停、气源、电源和安全状态，再检查 PLC 输出条件、阀线圈、气路压力、节流阀和机械卡滞。排查结果需要形成现象、证据、原因、处理和复测记录。

本材料仅用于 MVP 演示，不能替代真实企业访谈或正式岗位调研。后续应由教师导入真实岗位 JD、企业任务书、行业标准或课程标准后生成待确认建议。
