# Product Scope

## 一句话定位

一个面向机电一体化职业新人的岗位任务培训 MVP：能基于岗位画像诊断薄弱能力，推荐学习路径，生成实训任务卡与教师/师傅反馈。

## 目标用户

- 职业新人：快速知道自己距离“自动化生产线装调与运维技术员”岗位要求还差哪些能力。
- 高职实训学生：在真实岗位任务语境中完成传感器、PLC、气动和故障诊断训练。
- 教师/企业师傅：快速查看共性薄弱点，并获得可执行的补救教学或带教建议。
- 竞赛评审：能看到从诊断、推理、路径推荐到反馈闭环的完整样例。

## MVP 范围

包含：

- 意图分类
- 岗位画像
- 问答式主对话
- 岗位任务到学习任务映射
- 问题直答与现场证据追问
- 问题到能力/知识缺口映射
- 诊断问答
- 基于规则的评分
- 能力节点画像
- 知识点与资源推荐
- 训练任务推荐
- 一节课内可完成的实训任务卡
- 教师视角汇总
- 本地 Web/API Demo
- 可选 OpenAI-compatible 大模型接入
- 可选平台工作流复制配置

不包含：

- 登录注册
- 班级真实数据后台
- 完整 LMS 集成
- 自动批改复杂图纸或真实 PLC 程序
- 长期学习档案数据库
- 重型向量数据库或图数据库

## 核心指标

- 新人能在 3 分钟内完成一次岗位任务诊断。
- 每次诊断输出至少 3 个能力节点评分。
- 每个薄弱节点给出 1 个知识点、1 个训练任务、1 个资源和 1 条安全提醒。
- 教师/师傅视图能指出 Top 3 薄弱点和下一节课/下一次带教建议。

## MVP 输入输出

输入：

- 岗位画像或目标岗位
- 新人聊天问题、追问、故障现象或学习目标
- 传感器动作灯、PLC 输入灯、在线监控等现场状态
- 诊断题答案
- 可选：教师上传的班级答题摘要

输出：

- 意图类型
- 岗位任务定位
- 聊天回答、安全提醒和关键追问
- 高亮能力节点和知识缺口卡片
- 诊断得分与薄弱点
- 常见错误标签
- 个性化学习路径和实训任务卡
- 教师/师傅反馈摘要
